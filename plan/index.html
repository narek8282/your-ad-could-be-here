Visual Generation

Unified photo and video style

The visual system should feel like a premium production desk, not generic AI art: dark cinematic rooms, sponsor slots, phone screens, calendars, film frames, US holidays, and measurable demand signals. The image above is the first generated reference asset for that direction.

Master Image Prompt

Create a cinematic, premium visual for a film-content sales engine: dark production room, planning wall, sponsor cards, calendar blocks, phone screens for Stories/Reels, X and Reddit, subtle US holiday props, glowing empty ad-slot frames, red/gold/charcoal palette, realistic textures, no readable text, no fake logos.

Master Video Prompt

12-second cinematic motion: slow push-in across a dark production desk, film frames become glowing ad slots, phone screens animate Stories/Reels/X/Reddit cards, holiday campaign objects appear as the calendar advances, ending on an empty premium slot ready to be claimed. No voiceover unless added later.

Holiday Variant Rule

Keep the same room, lens, lighting, and palette. Change only the seasonal layer: Labor Day, Halloween, Black Friday, Christmas, New Year, Valentine's Day, July 4. This keeps every campaign recognizable as one brand.

Prompt System

What each content type must contain

Stories

3-5 slides: tension, project proof, open slot, one reply keyword.

Prompt: Turn this idea into 4 IG Story slides for US sponsors. Slide 1 must create curiosity, slide 2 explains the absurd film mechanic, slide 3 shows the paid place, slide 4 asks for one reply keyword.

Reels

15-25 seconds: first-frame hook, visual contradiction, slot reveal, CTA.

Prompt: Write a 20-second Reel script for a film made from ads. Include timestamped shots, voiceover, on-screen text, and a CTA to request a slot.

X

Daily public build log: sharp thesis, proof signal, open question, slot CTA.

Prompt: Write one X post and one 5-post thread about this project update. Make it credible to creators, sponsors, and investors. Include one data point or proof signal and end with a soft slot CTA.

Reddit

Discussion-first posts for film, advertising, startups, and weird internet communities.

Prompt: Write a Reddit post that starts a real discussion about a film built from ads. No hard selling. Include context, the creative/business question, what feedback is needed, and one transparent disclosure that the project is being built publicly.

Instagram Posts

One idea, short lines, proof or belief shift, slot CTA.

Prompt: Write an Instagram caption that makes sponsors understand why buying a visible place inside the film is culturally sharper than buying a normal ad.

Facebook Essays

Founder logic, market context, investor signal, detailed CTA.

Prompt: Write a Facebook long post explaining the business logic behind selling the film before production. Make it credible to sponsors, early supporters, and investors.

Investor Narrative

This plan is not only marketing. It is demand validation.

Each month should produce evidence: DMs, reservations, sponsor objections, price resistance, creative reactions, comments, and repeatable hooks. The investor story is simple: the project is testing distribution and revenue before production cost scales.

Monthly proof pack

  • Best performing hook
  • Most requested slot type
  • Number of qualified sponsor conversations
  • Reservation value and close rate
  • Audience objections converted into FAQ
  • Content that should become film material
