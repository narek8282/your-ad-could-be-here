# Your Ad Could Be Here

Simple 2D landing page for an anti-film built from ads, sponsor slots, and paid messages.

## Hosting

The site runs for free on GitHub Pages.

- Custom domain: `youradcouldbehere.com`
- GitHub Pages source: `main` / `/root`
- Paid hosting: not needed

## Files

- `index.html` - single-page landing page
- `styles.css` - responsive 2D design
- `assets/hero-anti-film.png` - cinematic hero image
- `CNAME` - custom domain for GitHub Pages

## Contact Flow

The MVP uses free contact paths instead of a backend:

- email link with prefilled request text
- Telegram link
